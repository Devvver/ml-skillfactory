Подстановки Эйлера Подстановки Эйлера — подстановки, приводящие интегралы вида     ∫ R ( x ,   a  x  2   + b x + c   ) d x   {\displaystyle \int R(x,{\sqrt {ax^{2}+bx+c}})dx}  , где     R ( x ,   a  x  2   + b x + c   )   {\displaystyle R(x,{\sqrt {ax^{2}+bx+c}})}   — рациональная функция, к интегралам от рациональных функций. Предложены Л. Эйлером в 1768 году[1][2]. Производится замена:       a  x  2   + b x + c   = ± x t ±   c     {\displaystyle {\sqrt {ax^{2}+bx+c}}=\pm xt\pm {\sqrt {c}}}    Третья подстановка[править | править код] Используется тогда, когда подкоренное выражение имеет два действительных корня. Производится замена:       a  x  2   + b x + c   = ± t ( x − λ )   {\displaystyle {\sqrt {ax^{2}+bx+c}}=\pm t(x-\lambda )}   , где     λ   {\displaystyle \lambda }   — один из корней[1].

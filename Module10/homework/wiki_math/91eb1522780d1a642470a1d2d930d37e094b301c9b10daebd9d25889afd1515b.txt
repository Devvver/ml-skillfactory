Цепная гомотопия Цепна́я гомото́пия — вариация понятия «гомотопия» в алгебраической топологии и гомологической алгебре  Определение[править | править код] Пусть     C   {\displaystyle C}   — цепной комплекс модулей (то есть семейство модулей      C  n     {\displaystyle C_{n}}   и модульных гомоморфизмов      d  n   :  C  n   →  C  n − 1     {\displaystyle d_{n}\colon C_{n}\to C_{n-1}}  ),     f   {\displaystyle f}   и     g   {\displaystyle g}   — цепные отображения комплекса     C   {\displaystyle C}   в комплекс      C ′    {\displaystyle C'}   (то есть такие гомоморфизмы      f  n     {\displaystyle f_{n}}   что      d  n    f  n   =  f  n − 1    d  n     {\displaystyle d_{n}f_{n}=f_{n-1}d_{n}}  ). Цепной гомотопией между отображениями     f   {\displaystyle f}   и     g   {\displaystyle g}   называется такое семейство гомоморфизмов      s  n   :  C  n   →  C  n + 1  ′    {\displaystyle s_{n}\colon C_{n}\to C'_{n+1}}  , что       s  n − 1    d  n   +  d  n + 1  ′   s  n   =  f  n   −  g  n   .   {\displaystyle s_{n-1}d_{n}+d'_{n+1}s_{n}=f_{n}-g_{n}.}    Свойства[править | править код] Если отображения     f   {\displaystyle f}   и     g   {\displaystyle g}   цепно гомотопны, то индуцированные отображения на гомологиях      H  n   ( C ) →  H  n   (  C ′  )   {\displaystyle H_{n}(C)\to H_{n}(C')}   равны (где      H  n   ( C ) =  K e r    d  n    /   I m    d  n + 1     {\displaystyle H_{n}(C)=\mathrm {Ker} \,d_{n}/\mathrm {Im} \,d_{n+1}}  ). Тогда      d  n   ( c ) = 0   {\displaystyle d_{n}(c)=0}  . Для большинства теорий гомологий доказывается, что гомотопные непрерывные отображения топологических пространств     f , g : X → Y   {\displaystyle f,g\colon X\to Y}   индуцируют цепно гомотопные отображения комплексов     C ( X ) → C ( Y )   {\displaystyle C(X)\to C(Y)}   и, по доказанному, одинаковые отображения групп гомологий     H ( X ) → H ( Y )   {\displaystyle H(X)\to H(Y)}   (выполняется аксиома гомотопической инвариантности).

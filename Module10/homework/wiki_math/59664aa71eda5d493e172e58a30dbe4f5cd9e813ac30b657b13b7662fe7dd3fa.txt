K-ядро K-ядро (от англ. kernel) — принцип оптимальности в кооперативных играх, впервые введен в работе М. Дэвиса и М. Машлера (1965). Пусть задана кооперативная игра с характеристической функцией     ν :  2  N   →  R    {\displaystyle \nu :2^{N}\to \mathbb {R} }   и     x ∈   R   N     {\displaystyle x\in \mathbb {R} ^{N}}   — эффективный вектор выигрышей. Максимальный излишек игрока     i   {\displaystyle i}   над игроком     j   {\displaystyle j}   по отношению к     x   {\displaystyle x}   определяется как      s  i j   ν   ( x ) = max  {  ν ( S ) −  ∑  k ∈ S    x  k   : S ⊆ N ∖ { j } , S ∋ i  }    {\displaystyle s_{ij}^{\nu }(x)=\max \left\{\nu (S)-\sum _{k\in S}x_{k}:S\subseteq N\setminus \{j\},S\ni i\right\}}  . Максимальный излишек представляет собой наибольший выигрыш, который игрок     i   {\displaystyle i}   может получить, войдя в какую-либо частичную коалицию     S   {\displaystyle S}   без кооперации с игроком     j   {\displaystyle j}  , в предположении, что остальные игроки в составе коалиции     S   {\displaystyle S}   удовлетворены выигрышами, которые доставляет им распределение     x   {\displaystyle x}  . Он представляет собой способ измерения сравнительной переговорной силы игроков. K-ядром кооперативной игры     ν   {\displaystyle \nu }   называется множество дележей     x   {\displaystyle x}  , удовлетворяющих условиям:     (  s  i j   ν   ( x ) −  s  j i   ν   ( x ) ) (  x  j   − ν ( j ) ) ≤ 0   {\displaystyle (s_{ij}^{\nu }(x)-s_{ji}^{\nu }(x))(x_{j}-\nu (j))\leq 0}  ;     (  s  j i   ν   ( x ) −  s  i j   ν   ( x ) ) (  x  i   − ν ( i ) ) ≤ 0   {\displaystyle (s_{ji}^{\nu }(x)-s_{ij}^{\nu }(x))(x_{i}-\nu (i))\leq 0}  ; для всех пар игроков     i , j   {\displaystyle i,j}  . Интуитивно, игрок     i   {\displaystyle i}   имеет большую переговорную силу, чем игрок     j   {\displaystyle j}   при дележе     x   {\displaystyle x}  , если      s  i j   ν   ( x ) >  s  j i   ν   ( x )   {\displaystyle s_{ij}^{\nu }(x)>s_{ji}^{\nu }(x)}  , но игрок     j   {\displaystyle j}   защищен от угроз игрока     i   {\displaystyle i}  , если      x  j   = ν ( j )   {\displaystyle x_{j}=\nu (j)}  , так как в этом случае он может получить выигрыш      x  j     {\displaystyle x_{j}}   без кооперации. K-ядро содержит все дележи, при которых ни один игрок не имеет такой переговорной силы ни над каким другим игроком.  Ссылки[править | править код] Davis, M., Maschler, M.

Трезубец (кривая)  трезубец с a = b = c = d = 1  трезубец с a = b = c = d = 1, окрестность точки y = ∞ В математике трезубец (а также трезубец Ньютона или парабола Декарта) — произвольный член семейства кривых, имеющих формулу:      x y + a  x  3   + b  x  2   + c x = d ,   {\displaystyle xy+ax^{3}+bx^{2}+cx=d,}   или     x y = P ( x ) ,   {\displaystyle xy=P(x),}   где     P ( x )   {\displaystyle P(x)}   — многочлен третьей степени. Трезубцы являются кубическими плоскими кривыми. На вещественной проективной плоскости они имеют обычную двойную точку x = 0, y = 1, z = 0. Если мы подставим x = x/z и y = 1/z в уравнение трезубца, получим      a  x  3   + b  x  2   z + c x  z  2   + x z = d  z  3   ,   {\displaystyle ax^{3}+bx^{2}z+cxz^{2}+xz=dz^{3},}   и эта кривая имеет обычную двойную точку в начале координат. Трезубцы, таким образом, являются рациональными плоскими кривыми с геометрическим родом нуль.  Литература[править | править код] J. Dennis Lawrence.

Метод перевала Метод перевала — метод, использующийся для аппроксимации интегралов вида       ∫  γ   Φ ( z )  e  λ ϕ ( z )    d z ,   {\displaystyle \int \limits _{\gamma }\Phi (z)e^{\lambda \phi (z)}\,dz,}   где     Φ ( z ) , ϕ ( z )   {\displaystyle \Phi (z),\phi (z)}   — некоторые мероморфные функции,     λ   {\displaystyle \lambda }   — некоторое большое число, а контур     γ ∈  C    {\displaystyle \gamma \in \mathbb {C} }   может быть бесконечным. Этот метод часто называется обобщением метода Лапласа. Из условий Коши — Римана следует, что в точках перевала кривые наискорейшего роста и наискорейшего убывания пересекаются под прямым углом, причём нигде кроме точек перевала они больше пересекаться не могут. Из этих простых соображений можно однозначно их построить. Кривые наискорейшего убывания представлены на рисунке (стрелками обозначено направление роста). Для того, чтобы воспользоваться методом Лапласа для нахождения асимптотики этого интеграла, необходимо линейными преобразованиями деформировать контур     γ   {\displaystyle \gamma }   по кривым наискорейшего убывания. Поскольку на этих кривых достигается глобальный максимум функции     ϕ ( s )   {\displaystyle \phi (s)}  , то мы можем рассматривать только небольшую его окрестность.
